# 生態保育措施參考照片放置與命名說明

網站會依 `src/content/ecoplan.md` 的「### 參考照片」清單讀取圖片，不需修改 TypeScript 程式。

## 放置步驟

1. 將照片裁切為橫式，建議比例 `4:3`、寬度至少 1200 像素。
2. 將照片轉存為 WebP 格式；副檔名使用小寫 `.webp`。
3. 以「措施名稱-兩位數序號.webp」命名，例如 `重要棲地保留-01.webp`、`重要棲地保留-02.webp`。
4. 將圖檔放入 `public/images/ecological-measures/`。
5. 開啟 `src/content/ecoplan.md`，在對應措施的 `### 參考照片` 下新增一行：`- 照片說明：重要棲地保留-02.webp`。
6. 執行 `npm run dev`，開啟網站的「生態保育措施」，切換到對應頁籤確認圖片與說明。

## 命名規則

- 「措施名稱」必須與 `ecoplan.md` 的 `##` 標題完全相同，序號從 `01` 開始依序增加。
- 可使用中文、數字與半形連字號 `-`；請勿加入空格。
- 不可使用 `/`、`\\`、`:`、`：`、`*`、`?`、`"`、`<`、`>`、`|` 等檔名不安全字元。
- Markdown 內的檔名（冒號右側）必須與實際圖檔完全相同，包含大小寫與副檔名。
- 冒號左側是網站顯示的中文照片說明，可自由修改，不會影響圖檔路徑。

若圖檔尚未放置或檔名不一致，頁面會顯示「尚未放置」提示，不會出現破圖圖示。
