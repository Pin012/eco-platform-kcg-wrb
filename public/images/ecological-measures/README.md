# 生態保育措施參考照片放置與命名說明

頁面文字與照片清單只需維護 `src/content/ecoplan.md`，圖片檔案放在本資料夾：

`public/images/ecological-measures/`

## 操作順序

1. 先準備 JPG、PNG 或 WebP 圖片；建議使用 WebP，橫式比例約 4:3，寬度至少 1200 像素。
2. 使用中文措施名稱重新命名，格式為「`措施名稱-兩位數流水號.webp`」，例如：
   - `重要棲地保留-01.webp`
   - `重要棲地保留-02.webp`
   - `生態友善照明-01.webp`
3. 檔名不可含有 `/`、`\`、`:`、`*`、`?`、`"`、`<`、`>`、`|`；同一措施的流水號由 `01` 依序增加。檔名、中文字與副檔名大小寫都必須和 Markdown 完全一致。
4. 將圖片複製到 `public/images/ecological-measures/`，不要建立措施子資料夾。
5. 開啟 `src/content/ecoplan.md`，在對應的 `### 參考照片` 下新增一行：

   ```md
   - 重要棲地保留-01.webp｜保留既有濱溪植被帶現況
   ```

   `｜` 前方是實際圖片檔名，後方是顯示於照片下方的中文說明。每張照片各寫一行；不需要照片時，保留空白的 `### 參考照片` 即可。
6. 在專案根目錄執行 `npm run dev`，開啟終端機顯示的網址，進入「生態保育措施」確認圖片與說明。

若圖片尚未放入或檔名不一致，頁面會顯示圖片預留圖示；請先比對本資料夾中的檔名與 Markdown 的檔名是否逐字相同。
